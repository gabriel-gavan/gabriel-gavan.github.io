🎮 Submit Your Game






Neon Mini Game Hub is building a fast, lightweight browser arcade focused on modern HTML5 experiences. We aim to create a curated collection of instant-play games that work smoothly on desktop and mobile — no installs, no accounts.

🚀 Our Vision

We are building a performance-focused browser arcade powered by:

  • ⚡ Pure HTML, CSS & JavaScript
  • 🎨 Clean neon UI design
  • 📱 Mobile-first responsive layouts
  • 🧠 Tight gameplay loops & replayability
  • 🔒 Minimal tracking & clean UX

We review each submission manually to ensure quality, performance, and originality.

🛠 What We Accept

We accept any game built with any technology — as long as it runs directly in the browser.

  • ✔ Any engine (Unity WebGL, Phaser, Three.js, Godot Web export, custom JS, etc.)
  • ✔ Any framework or build system
  • ✔ Must include a working index.html entry file
  • ✔ Must run in a standard modern browser
  • ✔ ZIP file containing the complete playable build
  • ✔ No malicious or hidden external scripts
  • ✔ Original work or properly licensed assets

The only requirement: the game must launch by opening index.html in a browser.
Maximum ZIP size: 50MB.

📦 How To Prepare Your ZIP

Your ZIP should contain:

  • index.html (entry file)
  • All JS/CSS files included
  • Assets folder if needed
  • No broken local paths

Make sure the game runs by simply opening index.html. If it requires a server (Node, PHP, etc.), please mention it in the description.

⏱ Review Process

All submissions are manually reviewed. Estimated review time: 24–72 hours.

We evaluate:

  • ⚡ Performance & load speed
  • 🎮 Gameplay quality & clarity
  • 📱 Mobile compatibility
  • 🔒 Security & safe scripting
  • 🎨 Overall polish

If your game meets our quality standards, we will contact you via email.
