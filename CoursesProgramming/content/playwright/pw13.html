Visual checks

// Snapshot a component
await expect(cmp).toHaveScreenshot('button.png');

Best practices

  • Keep components pure where possible; inject state via props.
  • Mock event handlers and assert outcomes rather than internal state.
  • Stabilize fonts and data when taking screenshots.

Exercises

  1. Mock props and verify event callbacks with spies.
  2. Test hover/focus/disabled states with screenshot comparison.
Tip: Use component tests for edge cases difficult to reach via E2E (error banners, validation messages, empty states).
