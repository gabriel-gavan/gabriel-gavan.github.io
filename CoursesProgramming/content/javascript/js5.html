Selecting and Updating the DOM

Manipulate elements by changing text, attributes, classes, and styles. Batch DOM reads/writes to avoid layout thrashing.

Select and update

const title = document.querySelector('#title');
title.textContent = 'Hello DOM';
title.classList.add('highlight');

document.querySelector('#name').setAttribute('aria-invalid', 'false');

Dataset and ARIA

const el = document.querySelector('[data-testid="price"]');
el.dataset.currency = 'USD';
el.setAttribute('aria-live', 'polite');

Efficient updates

// Use DocumentFragment to insert many nodes at once
const ul = document.querySelector('#list');
const frag = document.createDocumentFragment();
for (let i = 0; i < 100; i++) {
  const li = document.createElement('li');
  li.textContent = `Item ${i}`;
  frag.appendChild(li);
}
ul.appendChild(frag);

Templates

// HTML
// <template id="row-tpl"><li class="row"></li></template>

const tpl = document.querySelector('#row-tpl');
const li = tpl.content.firstElementChild.cloneNode(true);
li.textContent = 'Row';
document.querySelector('#rows').appendChild(li);
