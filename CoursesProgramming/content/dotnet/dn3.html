Control Structures in C#

Branch and loop to express program logic clearly. Prefer early returns and modern pattern matching.

if / else

if (age >= 18) Console.WriteLine("Adult");
else Console.WriteLine("Minor");

Switch expressions with patterns

string Describe(object o) => o switch
{
  null => "null",
  int n when n < 0 => "negative int",
  int n => $"int {n}",
  string s => $"string of length {s.Length}",
  _ => "unknown"
};

Loops

for(int i=0;i<3;i++) Console.WriteLine(i);

foreach (var s in new[]{"a","b","c"}) Console.WriteLine(s);

int n = 3; while (n-- > 0) { /* ... */ }
