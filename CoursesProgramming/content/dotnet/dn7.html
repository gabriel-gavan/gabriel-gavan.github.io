Lists, Dictionaries, and foreach

Choose the right collection for the job and iterate clearly. Prefer interfaces (IList<T>, IDictionary<TKey,TValue>).

Common collections

var list = new List { "a", "b" };
var set = new HashSet { 1, 2, 2 }; // {1,2}
var dict = new Dictionary { ["Ava"] = 10 };

// Lookup with TryGetValue (avoid KeyNotFoundException)
if (dict.TryGetValue("Ava", out var score)) { Console.WriteLine(score); }

Ordering and projections

var users = new[]{ new { Name="A", P=2 }, new { Name="B", P=5 } };
var ordered = users.OrderByDescending(u => u.P).ThenBy(u => u.Name).ToList();
var names = users.Select(u => u.Name).ToArray();

Immutable collections

// System.Collections.Immutable
a = a.Add(3); // returns a new collection

Exercises

  1. Write a function that counts word frequency using a dictionary.
  2. Sort a list of users by points, breaking ties by name.
  3. Prototype a read‑only view by exposing IReadOnlyList<T> from a class.
Tip: Expose read‑only interfaces to reduce accidental mutation. Use TryGetValue in hot paths to avoid double lookups.
