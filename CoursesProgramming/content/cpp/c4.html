Functions and Parameters

Break programs into small, reusable pieces. Understand pass‑by‑value vs references and const‑correctness.

Examples

void inc(int &x){ x++; }
int sum(int a, int b) { return a + b; }
int sum3(int a, int b, int c = 0) { return a + b + c; } // default arg

// Prefer const& for large objects to avoid copies
int size_of(const std::string& s) { return static_cast(s.size()); }

// constexpr (evaluated at compile-time when possible)
constexpr int sq(int n){ return n*n; }
static_assert(sq(3) == 9);

noexcept and return by value

std::string make_label(int n) noexcept { return "item-" + std::to_string(n); }

Exercises

  1. Write a function that swaps two integers using references.
  2. Refactor a function to take const std::string& instead of by value.
  3. Add a constexpr function and use static_assert to verify it.
Tip: Prefer passing small trivially copyable types by value; pass large or non‑trivial types by const&.
