The STL: Vectors, Maps, Iterators

The C++ Standard Library provides containers and algorithms. Combine them to write expressive, efficient code.

Vectors and capacity

#include 
std::vector v{1,2,3};
v.reserve(10); // avoid reallocations
v.push_back(4);
for(int n : v) { /*...*/ }
