Smart Pointers

Manage dynamic memory with RAII helpers. Use unique_ptr for exclusive ownership, shared_ptr for shared ownership, and weak_ptr to break cycles.

unique_ptr

#include 
auto p = std::make_unique(42);
// transfer ownership
std::unique_ptr q = std::move(p); // p becomes null

shared_ptr and weak_ptr

#include 
auto sp = std::make_shared("hi");
auto sp2 = sp; // refcount++
std::weak_ptr wp = sp; // non-owning; lock() to use
if (auto locked = wp.lock()) { /* use *locked */ }

Custom deleters

auto fileCloser = [](FILE* f){ if(f) std::fclose(f); };
using FilePtr = std::unique_ptr;
FilePtr f(std::fopen("data.txt", "r"), fileCloser);

Exercises

  1. Refactor a raw new allocation to unique_ptr.
  2. Use weak_ptr to break a cycle between two shared_ptr owners.
  3. Create a unique_ptr with a custom deleter for a FILE handle.
Tip: Prefer make_unique/make_shared over direct new for exception safety and clarity.
