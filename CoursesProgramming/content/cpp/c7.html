Constructors and Methods

Use initializer lists, delegating constructors, and explicit to avoid unintended conversions.

Examples

class Point {
  int x, y;
public:
  Point(): x(0), y(0) {}
  Point(int x, int y): x(x), y(y) {}
  explicit Point(int v): x(v), y(v) {} // avoid implicit one-arg conversions
  // Delegating constructor
  Point(double angle, double r, bool polar): Point(static_cast(r*std::cos(angle)), static_cast(r*std::sin(angle))) {}
  int X() const { return x; }
  int Y() const { return y; }
};

std::ostream& operator<<(std::ostream& os, const Point& p){ return os << '(' << p.X() << ',' << p.Y() << ')'; }

Exercises

  1. Mark methods const when they don’t change state.
  2. Add a factory/static helper to construct from polar coordinates.
  3. Overload operator<< for a small type and print it.
Tip: Prefer explicit for single‑argument constructors to prevent surprising implicit conversions.
