Variables, Types, and I/O

C++ has value semantics by default. Understand primitive types, references, const, and stream I/O.

Examples

#include 
#include 
int age = 20;
double price = 19.99;
bool ok = true;
std::cout << age << "\n" << std::fixed << std::setprecision(2) << price << "\n";

References and const

int x = 1; int& ref = x; ref++; // x becomes 2
const std::string s = "hello"; // cannot modify binding
const std::string& view = s;    // bind to existing object without copying

auto and type deduction

  • Use auto when the type is obvious from the initializer or long to spell.
  • Avoid auto when it hides important semantics (e.g., signed/unsigned).

Reading input

#include 
std::string line; std::getline(std::cin, line);
std::istringstream iss(line);
int a; double b; iss >> a >> b;

string vs string_view
