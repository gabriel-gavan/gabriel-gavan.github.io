Control Flow: if, switch, and loops

Control flow structures decide what code runs and how many times it runs. Prefer early returns over deep nesting.

if / else

if (age >= 18) {
  System.out.println("Adult");
} else {
  System.out.println("Minor");
}

Enhanced switch (Java 14+)

String activity = switch (day) {
  case MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY -> "Work";
  case SATURDAY, SUNDAY -> "Rest";
  default -> throw new IllegalArgumentException("Invalid day");
};

Loops

// for
for (int i = 0; i < 3; i++) System.out.println(i);

// while
int n = 3; while (n-- > 0) System.out.println(n);

// for-each
for (var s : List.of("a","b","c")) System.out.println(s);

// break/continue with labels
outer: for (int i=0;i<3;i++) {
  for (int j=0;j<3;j++) {
    if (i*j > 2) break outer; // exit both loops
  }
}
