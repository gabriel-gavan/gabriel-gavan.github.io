Java: What It Is and How to Start

Java is a statically typed, object‑oriented language that runs on the JVM. It compiles source into bytecode that executes consistently across platforms, making it a reliable choice for servers, tooling, and Android.

Install and verify

# macOS (Homebrew)
brew install openjdk

# Windows (winget)
winget install EclipseAdoptium.Temurin.21.JDK

# Linux (Debian/Ubuntu)
sudo apt-get update && sudo apt-get install -y openjdk-21-jdk

# Verify
java -version
javac -version

Hello World

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, Java!");
  }
}

Project structure

  • Place source under src/ and compile to out/.
  • One public class per file; file name matches the class name.
  • Use packages (package com.example.app;) to group related classes.

Build tools (Maven/Gradle)

# Maven quickstart
mvn -q archetype:generate -DgroupId=com.example -DartifactId=hello -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false

# Gradle init
gradle init --type java-application
  • Maven: convention over configuration, uses pom.xml.
  • Gradle: flexible, Groovy/Kotlin DSL, uses build.gradle or build.gradle.kts.

Run and package

# Compile & run (javac/java)
javac -d out src/HelloWorld.java && java -cp out HelloWorld

# Maven
mvn -q test && mvn -q package

# Gradle
gradle test && gradle build

Exercises

  1. Create a Main class that prints your name and the current year.
  2. Compile with javac and run with java.
  3. Initialize a Maven or Gradle project and run a simple unit test.
Tip: Use an LTS JDK (e.g., 21) for production; experiment with latest features on a separate toolchain.
