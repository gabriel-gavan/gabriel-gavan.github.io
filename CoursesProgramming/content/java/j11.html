Unit Testing with JUnit

Fast unit tests give rapid feedback and enable refactoring. Use JUnit 5 (Jupiter).

Basics

import static org.junit.jupiter.api.Assertions.*;
import org.junit.jupiter.api.*;

class MathTest {
  @BeforeEach void setUp() { /* runs before each */ }
  @AfterEach  void tearDown() { /* runs after each */ }

  @Test void adds(){ assertEquals(4, 2+2); }

  @Test void throwsOnBadInput(){
    IllegalArgumentException ex = assertThrows(IllegalArgumentException.class, () -> {
      throw new IllegalArgumentException("bad");
    });
    assertEquals("bad", ex.getMessage());
  }
}

Parameterized tests

import org.junit.jupiter.params.ParameterizedTest;
import org.junit.jupiter.params.provider.CsvSource;

class StringsTest {
  @ParameterizedTest
  @CsvSource({"ab,2", "hello,5"})
  void length(String input, int expected){
    assertEquals(expected, input.length());
  }
}

Exercises

  1. Test error cases using assertThrows.
  2. Add a parameterized test for edge cases.
  3. Adopt AAA (Arrange‑Act‑Assert) in your tests.
Tip: Test public behavior, not private implementation details. Keep tests fast and deterministic.
