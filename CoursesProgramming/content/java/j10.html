File I/O with NIO.2

Use Path and Files for modern I/O. Prefer UTF‑8 and try‑with‑resources.

Basics

import java.nio.file.*;
import java.nio.charset.StandardCharsets;

var p = Path.of("notes.txt");
Files.writeString(p, "Hello", StandardCharsets.UTF_8);
String s = Files.readString(p, StandardCharsets.UTF_8);

boolean exists = Files.exists(p);
long size = Files.size(p);

List, walk, copy

// List directory
try (var stream = Files.list(Path.of("."))) {
  stream.forEach(System.out::println);
}

// Walk tree and copy .txt files
try (var paths = Files.walk(Path.of("src"))) {
  paths.filter(x -> x.toString().endsWith(".txt")).forEach(System.out::println);
}

Reading lines

try (var lines = Files.lines(Path.of("data.csv"))) {
  lines.map(l -> l.split(","))
       .forEach(cols -> System.out.println(cols[0]));
}

Exercises

  1. List all files in a directory using Files.list.
  2. Copy a directory tree safely with Files.walk and Files.copy.
  3. Read a CSV file and build a list of POJOs.
Tip: Always close streams; prefer try‑with‑resources to avoid leaks.
