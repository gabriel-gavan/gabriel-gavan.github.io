Stream API Basics

Transform collections declaratively using streams. Streams are lazy; operations are evaluated when a terminal operation is invoked.

Pipeline: source → intermediate → terminal

var nums = List.of(1,2,3,4);
int sumSquares = nums.stream()
  .filter(n -> n % 2 == 0)
  .map(n -> n * n)
  .reduce(0, Integer::sum);

Common operations

  • Intermediate: filter, map, flatMap, distinct, sorted, peek
  • Terminal: forEach, collect, reduce, count, findFirst/findAny

Collectors

import static java.util.stream.Collectors.*;

var words = List.of("apple","apricot","banana");
var byFirst = words.stream().collect(groupingBy(w -> w.substring(0,1)));
var joined = words.stream().collect(joining(", "));
var toMap  = words.stream().collect(toMap(w -> w, String::length));

Optionals and streams

import java.util.Optional;
Optional maybe = Optional.of(42);
int val = maybe.stream().map(x -> x + 1).findFirst().orElse(0);

Exercises

  1. Filter even numbers and collect to a list.
  2. Group words by their first letter.
  3. Build a frequency map of characters in a string.
Tip: Avoid side effects inside stream operations; prefer pure transformations for readability and correctness.
